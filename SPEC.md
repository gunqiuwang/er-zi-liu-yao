# 二字·六爻 - 产品设计规格

## 1. 概念与愿景

一个极简、安静、带有东方留白气质的微信小程序。核心体验：

> 1. 每天输入两个字 → 2. 起卦获得完整卦象与原文 → 3. 年历记录

整个 App 像一个**数字签筒**，东方仪式，安静的情绪容器。

风格关键词：极简、留白、安静、呼吸感、东方感、克制、纸张感、非商业感。

---

## 2. 设计语言

### 2.1 美学方向
- **风格**: 东方极简禅意 + 和纸质感
- **参考**: HIBI 二字日记、签筒、展览海报、水墨画
- **核心感受**: 安静的数字器物，非工具感，非商业感

### 2.2 色彩系统

```css
/* 背景 - 带轻微和纸纹理 */
background: #F7F4EE;

/* 主文字 */
color: #1C1C1C;

/* 次级文字 */
color: #8B8B8B;

/* 点缀色 - 朱砂红（极少使用） */
color: #9F2D20;
```

朱砂红仅用于：小型印章、日期标记、极小细节。

### 2.3 字体系统

- **标题/正文**: Noto Serif SC / 思源宋体 / LXGW WenKai
- **数字**: SF Pro Display
- **整体偏宋体感**，不要现代科技字体

### 2.4 空间系统

- 页面边距: `48rpx`
- 元素间距: `32rpx` / `48rpx` / `64rpx`
- 大量留白，呼吸感
- 安全区域: 底部 `120rpx`

### 2.5 动效哲学

- 所有动画：**慢、轻、克制、有停顿**
- 页面切换: `400ms ease-out`
- 元素出现: `800ms` 淡入
- 铜钱点击: 微旋转 + 轻震动
- 卦象展开: 5秒缓慢序列
- **禁止**: 弹跳、炫光、粒子、游戏感

## 4. 首页设计

### 4.1 页面结构

```
┌─────────────────────────────────┐
│  乙巳年 · 孟夏 · 十九    [今日] │  ← 朱砂红小印记
│                                 │
│                                 │
│            「微雨」              │  ← 视觉中心，大字
│                                 │
│                                 │
│        ◉ [铜钱]                 │  ← 极简铜钱 icon
│                                 │
│           [分享]                │  ← iOS 风格分享 icon
│                                 │
│   [📅]                    [⚙️] │  ← 底部导航图标
└─────────────────────────────────┘
```

### 4.2 顶部日期区

- 中式农历日期横排：`乙巳年 · 孟夏 · 十九`
- 旁边极小朱砂红印记：`今日`
- 视觉安静，有呼吸感，类似展览海报

### 4.3 二字输入区

- 显示示例：`「微雨」`
- 字体明显大于其它区域
- 不要输入框边框
- 不要 placeholder
- 只保留极淡角标：`「 」`
- 整个页面视觉中心

### 4.4 铜钱按钮（核心交互）

**视觉样式**:
```css
width: 112rpx;      /* 56px */
height: 112rpx;     /* 56px */
border-radius: 50%;
background: radial-gradient(circle at top, #8B6B3F, #5B4630);
box-shadow: 0 4rpx 16rpx rgba(0,0,0,0.08);
```

**中间方孔**:
```css
width: 28rpx;
height: 28rpx;
background: #F7F4EE;
```

**点击效果**:
- 微旋转
- 轻震动 (wx.vibrateShort)
- 生成卦象

**不要普通按钮文字**

### 4.5 分享按钮

- 仅 icon，无文字
- iOS 风格分享 icon
- 极细线条，灰黑色
- 不要实体按钮感

## 5. 卦象显示页

### 5.1 交互方式

点击铜钱后，**不要跳转页面**，原地展开。

### 5.2 出现顺序（动画序列）

1. 停顿 `0.8s`
2. 第一爻出现
3. 第二爻出现
4. 第三爻出现
5. 完整卦象（第四至第六爻）
6. 卦名 + 原文

**整体过程**: `4~5秒`，缓慢，安静

### 5.3 卦象区域

```
        ䷴

      水泽节

  节，亨。苦节不可，贞。
```

- 卦象居中
- 大量留白
- 原文小字号
- **不做解释** - 不要 AI 分析、运势提示、白话翻译

---

### 6.1 结构

```
「微雨」

䷴

水泽节

```

## 7. 年历/卦历页面

### 7.1 设计方向

不是普通日历，更像**卦历**：

每一天只显示：
- 小点 / 小卦象 / 极简标记
- 形成时间纹理

### 7.2 点击日期展开

```
二字
卦象
原文
```

不要复杂交互。
